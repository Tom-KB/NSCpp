Insert a value at the given index in a serialized string.

Warning
The value at this index will be replaced by the one you gave.

Index is based on separator's split. E.g. : "A/B/C" with sep="/" => [0] = "A", [1] = "B", [2] = "C"

Parameters
indexIndex of the insertion.
separatorSeparator used in the serialized string.
seriStrSerialized string on which the insertion is made.
valueValue to insert at the given index.
