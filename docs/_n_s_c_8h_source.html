2Network System for C (NSC)
3Version 2.0
4Starring : Thomas K/BIDI

6Description:
7This library is a network system for C language.
8The library is cross-platform and can be used on Windows and Linux.
9It is based on the Winsock2 library for Windows and the socket library for Linux.
10It lets you create TCP/UDP servers and clients in IPv4 and IPv6.
11It also lets you send and receive data in a simple way.
